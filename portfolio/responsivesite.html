Responsive Website

At the begining of the semester I learned about the basics of HTML and CSS, which I used to design a website for a given style of art with mobile-first method, then made it responsive with CSS grid.

❮ ❯

The style I was given was Art Nouveau. I started the design process with resarch about the style. I made the website with proportions for mobile first. Then with the use of prototypes, style tile and Gestalt's design principles made a visual language that could be translated to desktop. These tools guided me throughout the whole process when making the website responsive.
